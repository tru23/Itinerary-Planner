Check out the live demo [here](https://itinerary-planner-rho.vercel.app/).

This is a simple web application built with React.js that demonstrates basic CRUD (Create, Read, Update, Delete) operations.

Features
Create: Add new items to the list.
Read: View the list of items.
Update: Edit existing items.
Delete: Remove items from the list.

start the application, run:

Using npm:

npm start
